Introduction to HTML

What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It describes the structure of a web page and consists of a series of elements that tell the browser how to display the content.

Think of HTML as the skeleton of a website. Just like how your bones give structure to your body, HTML gives structure to web content. It's the foundation upon which all websites are built.

Key Concept

HTML is not a programming language; it's a markup language. This means it uses special syntax to annotate (or "mark up") text to define how it should be structured and presented in a web browser.

The Evolution of HTML

HTML has evolved significantly since its creation in the early 1990s by Tim Berners-Lee. Here's a brief timeline of its development:

  • 1991: HTML 1.0 - The original version (never standardized)
  • 1995: HTML 2.0 - The first standard specification
  • 1997: HTML 3.2 - Added tables, applets, and text flow around images
  • 1999: HTML 4.01 - Introduced stylesheets, scripts, and frames
  • 2000-2014: XHTML - A stricter, XML-based version of HTML
  • 2014: HTML5 - The current standard with improved support for multimedia, new semantic elements, and APIs

Today, we use HTML5, which introduced many new features that make it easier to include audio, video, and interactive graphics on websites. It also introduced semantic elements that better describe the content they contain, making web pages more accessible and SEO-friendly.

Why Learn HTML?

HTML is the fundamental building block of the web. Here's why learning HTML is essential:

Foundation of Web Development

All web development starts with HTML. Even the most complex websites and web applications are built on HTML foundations.

Universal Language

HTML is supported by all browsers and devices that can access the web, making it truly universal.

Career Opportunities

Understanding HTML is a prerequisite for careers in web development, design, digital marketing, and content creation.

Content Control

Knowing HTML gives you precise control over how your content appears on the web.

How HTML Works

HTML uses a system of tags to define elements on a web page. These elements form a tree-like structure known as the Document Object Model (DOM).

When you visit a website, your browser:

  1. Requests the HTML file from the server
  2. Parses the HTML code
  3. Builds the DOM tree
  4. Renders the page according to the DOM structure

The Browser's Role

Different browsers may render the same HTML slightly differently. This is why web developers test their sites across multiple browsers to ensure consistency.

Your First HTML Element

Let's look at a simple HTML element to understand its structure:

<p>This is a paragraph of text.</p>

This example shows a paragraph element. It consists of:

  • An opening tag: <p>
  • The content: "This is a paragraph of text."
  • A closing tag: </p>

The opening and closing tags define where the element begins and ends. The content is what appears on the web page.

HTML and Other Web Technologies

HTML rarely works alone. Modern websites combine three core technologies:

HTML

Provides the structure and content of web pages.

CSS

Controls the presentation and styling of HTML elements.

JavaScript

Adds interactivity and dynamic behavior to web pages.

In this course, we'll focus on HTML and CSS, the two fundamental technologies that control the structure and appearance of web pages.

Summary

In this introduction, we've learned that:

  • HTML is a markup language used to structure content on the web
  • It has evolved significantly since its creation in the early 1990s
  • HTML5 is the current standard with improved features and semantic elements
  • HTML elements consist of opening tags, content, and closing tags
  • HTML works alongside CSS and JavaScript to create modern websites
